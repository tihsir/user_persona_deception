# PMIYC Collaboration Repo

This repository is set up for the Week 1 deliverables of our class project.

## Structure
- `src/`: Source code for the PMIYC framework (contains `pmiyc` and `runner`).
- `data/`: Contains `baseline_conversations` logs.
- `personas/`: Custom persona definitions (Person A's workspace).
- `results/`: Output results from experiments.
- `notebooks/`: Jupyter notebooks for analysis.
